# Запуск демо с WebTransport

### 1. Подготавливаем окружение
В терминале в папке с файлами создаем виртуальное окружение и ставим библиотеки:

```bash
python3 -m venv venv
source venv/bin/activate
pip install aioquic cryptography
pip3 install dotenv
cp .env-example .env
```

### 2. Генерируем сертификаты
Создаем SSL-сертификаты строго для IP `127.0.0.1` (чтобы избежать проблем с IPv6):

```bash
./gen_certs.sh
```

### 3. Запускаем сервер
В текущем терминале запускаем Python-скрипт (он должен висеть запущенным):

```bash
python3 server.py
```

### 4. Запускаем клиент (Chrome)
Открываем **новое окно терминала**. Выполняем команду для запуска Хрома без проверки сертификатов:

```bash
./run_chrome.sh
```

В открывшемся Chrome можно будет увидеть статус подключения.
